## 适配器接口

### TypeScript 定义

```typescript
// src/core/command-generation/types.ts

export interface ToolCommandAdapter {
  /** 工具标识（如 'claude', 'cursor'） */
  toolId: string;

  /**
   * 获取命令文件路径
   * @param commandId - 命令标识（如 'explore'）
   * @returns 相对于项目根目录的路径
   */
  getFilePath(commandId: string): string;

  /**
   * 格式化文件内容
   * @param content - 命令内容
   * @returns 完整文件内容（含 frontmatter）
   */
  formatFile(content: CommandContent): string;
}

export interface CommandContent {
  id: string;          // 命令标识
  name: string;        // 显示名称
  description: string; // 描述
  category: string;    // 分类
  tags: string[];      // 标签
  body: string;        // 命令主体
}
```

## 适配器示例：Claude

```typescript
// src/core/command-generation/adapters/claude.ts

import type { ToolCommandAdapter, CommandContent } from '../types.js';

export const claudeAdapter: ToolCommandAdapter = {
  toolId: 'claude',

  getFilePath(commandId: string): string {
    // Claude Code 使用 .claude/commands/opsx/ 目录
    return `.claude/commands/opsx/${commandId}.md`;
  },

  formatFile(content: CommandContent): string {
    // Claude 使用 YAML frontmatter
    const frontmatter = [
      '---',
      `description: ${content.description}`,
      '---',
    ].join('\n');

    return `${frontmatter}\n\n${content.body}`;
  }
};
```

## 适配器示例：Cursor

```typescript
// src/core/command-generation/adapters/cursor.ts

import type { ToolCommandAdapter, CommandContent } from '../types.js';

export const cursorAdapter: ToolCommandAdapter = {
  toolId: 'cursor',

  getFilePath(commandId: string): string {
    // Cursor 使用 .cursor/rules/ 目录
    return `.cursor/rules/openspec-${commandId}.md`;
  },

  formatFile(content: CommandContent): string {
    // Cursor 不使用 frontmatter
    return `---
globs:
  - "**/*"
---

# ${content.name}

${content.description}

${content.body}`;
  }
};
```

## 添加新工具适配器

### 步骤 1：创建适配器文件

```typescript
// src/core/command-generation/adapters/my-tool.ts

import type { ToolCommandAdapter, CommandContent } from '../types.js';

export const myToolAdapter: ToolCommandAdapter = {
  toolId: 'my-tool',

  getFilePath(commandId: string): string {
    return `.my-tool/commands/${commandId}.md`;
  },

  formatFile(content: CommandContent): string {
    return `# ${content.name}\n\n${content.description}\n\n${content.body}`;
  }
};
```

### 步骤 2：注册适配器

```typescript
// src/core/command-generation/adapters/index.ts

import { myToolAdapter } from './my-tool.js';

export const adapters: Record<string, ToolCommandAdapter> = {
  // ...existing adapters
  'my-tool': myToolAdapter,
};
```

### 步骤 3：添加到工具列表

```typescript
// src/core/config.ts

export const AI_TOOLS: AIToolOption[] = [
  // ...existing tools
  {
    value: 'my-tool',
    name: 'My Tool',
    skillsDir: '.my-tool/skills',
    commandsDir: '.my-tool/commands',
  },
];
```

## Skills vs Commands

### Skills（推荐）

```
.claude/skills/openspec-apply/
├── SKILL.md          # 技能定义
└── skill/
    └── instruction.md
```

**特点**：
- 支持更多元数据
- 可以包含多个文件
- 更好的组织结构

### Commands（传统）

```
.claude/commands/openspec/
├── propose.md
├── apply.md
└── archive.md
```

**特点**：
- 简单直接
- 兼容性好
- 适合简单场景

### 切换方式

```bash
# 设置为 skills only
openspec config delivery skills

# 设置为 commands only
openspec config delivery commands

# 两者都有
openspec config delivery both
```
